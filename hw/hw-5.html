2. Consider the "JavaScript Where To" lesson, linked from "JS Where To." JavaScript may be placed within HTML tags. Let us call these code passages "scripts." Can a script be placed in an HTML head, body, neither, or both?

A script can be placed in either the HTML <head> or the <body> section. It may be written inline within the page or linked externally using the <script> tag. While scripts can appear in both sections, larger external scripts are often placed just before the closing </body> tag to improve page loading performance.
